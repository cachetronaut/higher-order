# Common Mistakes and Filler Words

## Common Mistakes

| Anti-pattern | Fix |
|--------------|-----|
| Combining a condition and an action in one sentence | Separate into two sentences |
| Writing "the field" without naming it | Name it exactly, in backticks |
| Leaving a branch unstated | Write "do nothing; continue to the next step" if nothing happens |
| Using jargon in summaries or descriptions | Substitute plain equivalents |
| Using idioms, metaphors, or cultural references | State the literal meaning |
| Saying "there is an issue with" | State what the issue IS |
| Pasting full specs into chat messages | Link to the document instead |

## Filler Words to Avoid

| Avoid | Use instead |
|-------|-------------|
| leverage | use |
| utilize | use |
| in order to | to |
| going forward | from now on |
| at this time | now |
| as discussed / per our conversation | (omit) |
| deep dive | review |
| circle back | follow up |
| on the same page | agreed |
| move the needle | improve |
| low-hanging fruit | easiest fix |
| approximately | about |
| the team | (name the team) |
